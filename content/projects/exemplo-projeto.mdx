## O problema

Portfolios de dev costumam ser ou muito simples (um README glorificado) ou excessivamente complexos...

## A solução

Um template opinado, com stack moderna e design system próprio, que qualquer dev pode forkar e customizar em menos de uma hora...

## Decisões técnicas

### Velite em vez de Contentlayer

Contentlayer foi descontinuado. Velite é a alternativa mais madura com suporte ativo...

### Tailwind v4

A nova API de CSS variables facilita theming sem precisar de um arquivo de config separado...

## Resultado

Site no ar, template disponível para fork, README com instruções completas.
