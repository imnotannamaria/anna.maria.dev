## The problem

Developer portfolios tend to be either too simple (a glorified README) or overly complex with frameworks overkill for a personal site.

## The solution

An opinionated template with a modern stack and a custom design system, forkable and customizable in under an hour.

## Technical decisions

### Velite instead of Contentlayer

Contentlayer was discontinued. Velite is the most mature alternative with active support and Turbopack compatibility.

### Tailwind v4

The new CSS variables API makes theming straightforward without a separate config file. Design tokens defined once, available everywhere as Tailwind utilities.

## Result

Site live, template available to fork, README with full setup instructions.
