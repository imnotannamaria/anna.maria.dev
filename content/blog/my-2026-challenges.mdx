I don't usually do goal posts. But this year feels different, so I'm writing it down.

## What I'm building toward

I started 2026 working on [resend-ecommerce](https://github.com/imnotannamaria/resend-ecommerce), a project I built specifically to get deeper into the Resend ecosystem. It reminded me how much I enjoy building things from scratch with no deadline and no stakeholder asking for a status update.

So I set two challenges for myself this year.

## The challenges

**12 open source projects**

One per month. Some will be small tools, some will be bigger. The goal is not to go viral, it's to ship consistently and build things I actually want to exist.

`1/12 open source projects this year`

**12 open source contributions**

Not just my own repos. I want to contribute to projects I use and respect. A bug fix, a feature, a docs improvement. Anything that ships.

`0/12 open source contributions this year`

## Why I'm doing this

I've been heads down in product work for a few years now. Good work, real impact, but mostly behind closed doors. This year I want to build in public, get comfortable shipping imperfect things, and hopefully connect with other developers along the way.

I'll write about each project here as I go.

Let's see how it goes.
